Frequently Asked Questions

Find answers to your most frequently asked questions.

When a web page is loaded, the browser creates a Document Object Model(DOM) of the page. The Document Object Model(DOM) is a programming interface for web documents. It is responsible for managing the structure and behavior of web pages. It allows script to update, access and interact with the HTML or XML content dynamically.

We can select an element from the DOM by using getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector() and querySelectorAll().

Event delegation is a javascript design pattern used to efficiently manage and handle events on multiple child elements by attaching a single event listener to a common parent element. Event delegation is useful because it improves both performance and flexibility when handling events in the DOM.

We can manipulate an element's attributes by using setAttribute(), getAttribute() and removeAttribute() and styles like element.classList.add() or element.classList.remove() or element.style.color using the DOM.
